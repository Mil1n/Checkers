# Идеи для развития «Шашек»

## Уже добавлено в MVP

- **Гроссмейстерский штаб после каждого хода**: оценка, последний ход, главный ответ соперника, план и топ-кандидаты.
- **Анти-зевок**: выбранный ход проверяется на опасный тактический ответ соперника.
- **Тренер по стилю игры**: после ходов игрок получает базовый профиль — тактик, позиционщик или эндшпилист.
- **Режим “Спасти позицию”**: отдельная тренировочная позиция с задачей найти ресурс защиты.
- **Фото/скрин доски → анализ**: добавлена точка входа для загрузки изображения и будущего CV-распознавания.
- **Объяснимый бот**: топ-3 ходов показываются с оценками, а анализ объясняет материал, темп, центр и угрозы.
- **PDN/FEN слой**: история ходов экспортируется, позицию можно импортировать в FEN-подобном формате.
- **Редактор позиции**: кликом по клеткам можно собрать позицию для анализа.
- **Web Worker для бота**: глубокий поиск вынесен из основного UI-потока там, где браузер поддерживает Worker.

## Что есть у конкурентов

- Базовые приложения обычно закрывают «играть с ботом», онлайн-мультиплеер, блиц, задачи, undo, hints, профили, темы и несколько уровней сложности.
- Сильные платформы вроде lidraughts предлагают анализ, редактор доски, импорт/экспорт партий, турниры, статистику, задачи, варианты шашек и игры с друзьями.
- Простые браузерные проекты конкурируют низким порогом входа: без регистрации, быстрый старт, pass-and-play, daily challenge, force jump и сохранение партии.

## Следующие шаги

1. Подключить настоящее распознавание доски по фото: нормализация перспективы, классификация клеток, список неоднозначных фигур.
2. Сделать онлайн-комнаты через WebSocket/WebRTC с синхронизацией ходов и пост-партийным тренерским отчётом.
3. Добавить полноценный PDN-парсер, базу дебютов и объяснение дебютных планов человеческим языком.
4. Усилить движок: таблицы окончаний, iterative deepening, transposition table и ограничение по времени.
5. Добавить турнирные ничейные правила, часы, рейтинги, задачи дня и персональные тренировочные наборы.
