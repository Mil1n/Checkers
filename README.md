# ♟️ Checkers Grandmaster — русские шашки нового поколения

> **Не просто “шашки с ботом”, а персональный гроссмейстерский штаб прямо в браузере.**

Checkers Grandmaster — это MVP браузерной игры в русские шашки, где бот не молчит после хода, а объясняет позицию как тренер: показывает оценку, лучший план, опасные ответы соперника и топ-кандидатов. Цель проекта — сделать шашки онлайн не очередной доской с фишками, а умным тренажёром, который помогает реально становиться сильнее.

---

## 🚀 Что уже умеет MVP

### 🧠 Гроссмейстерский анализ

После ходов игра показывает:

- 📍 оценку позиции;
- 🏆 последний ход;
- 📖 понятное объяснение идеи;
- ⚠️ главную угрозу соперника;
- 🎯 план на ближайшие ходы;
- ♟️ топ-3 кандидатных хода с оценками.

### 🤖 Бот с поиском

Внутри есть собственный движок:

- обязательные взятия;
- максимальное взятие;
- многоходовые цепочки боя;
- дальнобойные дамки;
- превращение в дамку;
- оценка материала, центра и продвижения;
- alpha-beta поиск лучших ходов.

### 🛡️ Анти-зевок

Выбираешь шашку — игра проверяет, не получает ли соперник сильный тактический ответ. Это первый шаг к режиму, где приложение не даёт игроку “случайно проиграть за один ход”.

### 🔥 Режим “Спасти позицию”

Отдельный тренировочный сценарий: игрок получает неприятную позицию и должен найти ресурс защиты, встречную угрозу или путь к ничьей.

### 🧩 Редактор позиции

Можно включить редактор и кликами собрать свою позицию для анализа:

1. пустая клетка → белая шашка;
2. белая шашка → белая дамка;
3. белая дамка → чёрная шашка;
4. чёрная шашка → чёрная дамка;
5. чёрная дамка → пустая клетка.

### 📝 PDN/FEN-слой

MVP умеет:

- экспортировать историю ходов в PDN-подобном виде;
- сохранять текущую позицию в FEN-подобной строке;
- импортировать позицию обратно для анализа.

### 🧵 Web Worker для бота

Расчёт хода бота вынесен в Web Worker, чтобы глубокий поиск не подвешивал интерфейс. Если Worker недоступен, используется синхронный fallback.

### 📸 Фото доски — задел на будущее

В интерфейсе уже есть загрузка изображения. Сейчас это точка входа, а следующий этап — распознавание доски, проверка неоднозначных клеток и автоматический анализ позиции со скрина или фото.

### 🌐 Онлайн-комната — UX-прототип

Кнопка создаёт ссылку комнаты. Пока это локальный прототип, но архитектурно это место для будущей синхронизации через WebSocket или WebRTC.

## ✨ Почему это должно быть круче обычных “шашек онлайн”

Обычные проекты часто дают:

- 🎮 доску;
- 🤖 бота;
- 🌐 онлайн-комнату;
- ↩️ undo;
- 💡 подсказку;
- 🏆 рейтинги.

**Наша ставка — объяснимость и тренировка:**

- 🧠 бот рассказывает, почему ход сильный;
- 🛡️ анти-зевок предупреждает о тактике;
- 🔥 есть режим спасения плохих позиций;
- 🧩 можно собрать любую позицию;
- 📝 можно экспортировать и импортировать анализ;
- 📸 будет анализ по фото;
- 👤 профиль игрока показывает стиль игры.

## 🛠️ Быстрый старт

Проект не требует сборки и тяжёлых зависимостей.

```bash
npm test
```

Чтобы открыть игру, достаточно запустить любой статический сервер в корне репозитория, например:

```bash
python3 -m http.server 8080
```

И перейти в браузере:

```text
http://localhost:8080
```

> ⚠️ Для Web Worker лучше открывать проект через локальный сервер, а не напрямую через `file://`.

## 🧪 Тесты

Тесты проверяют:

- ✅ стартовую расстановку на 24 шашки;
- ✅ наличие легальных ходов;
- ✅ сохранение материала после тихого хода;
- ✅ генерацию кандидатных ходов бота;
- ✅ человекочитаемую нотацию;
- ✅ обязательное взятие;
- ✅ правило максимального взятия;
- ✅ FEN roundtrip;
- ✅ тренировочную позицию;
- ✅ формат предупреждения анти-зевка.

Запуск:

```bash
npm test
```
